Linear Partial Fractions

1. Why Partial Fractions?

Partial-fraction decomposition rewrites one complicated rational function as a sum of simpler rational functions whose antiderivatives are logarithms.

REQUIRED STARTING CONDITIONS

Before decomposing \(P(x)/Q(x)\):

  1. make the fraction proper, so \(\deg P<\deg Q\); and
  2. factor the denominator completely over the real numbers.

If the fraction is improper, perform polynomial long division first.

This section considers denominators that factor into linear terms.

2. Distinct Linear Factors

If \[ Q(x)=(a_1x+b_1)(a_2x+b_2)\cdots(a_nx+b_n) \] has distinct linear factors, assign one constant numerator to each factor.

DISTINCT LINEAR-FACTOR FORM

For example,

\[ \boxed{ \frac{P(x)}{(x-r)(x-s)} =\frac{A}{x-r}+\frac{B}{x-s} }, \qquad r\neq s. \]

Multiply both sides by the common denominator, then solve for the unknown constants by substituting convenient values or comparing coefficients.

EXAMPLE 1: TWO DISTINCT LINEAR FACTORS

Evaluate \[ \int\frac{5x+1}{(x-1)(x+2)}\,dx. \]

Write \[ \frac{5x+1}{(x-1)(x+2)} =\frac{A}{x-1}+\frac{B}{x+2}. \]

Multiplying by \((x-1)(x+2)\) gives \[ 5x+1=A(x+2)+B(x-1). \]

Set \(x=1\): \(6=3A\), so \(A=2\). Set \(x=-2\): \(-9=-3B\), so \(B=3\). Thus,

\[ \begin{aligned} \int\frac{5x+1}{(x-1)(x+2)}\,dx &=\int\left(\frac2{x-1}+\frac3{x+2}\right)dx\\[3pt] &=\boxed{2\ln|x-1|+3\ln|x+2|+C}. \end{aligned} \]

3. The Cover-Up Shortcut

For distinct linear factors, constants can often be found quickly by temporarily covering the corresponding factor.

EXAMPLE 2: THREE DISTINCT FACTORS

Decompose \[ \frac{2x^2+1}{x(x-1)(x+1)} =\frac{A}{x}+\frac{B}{x-1}+\frac{C}{x+1}. \]

Multiplying through gives \[ 2x^2+1=A(x-1)(x+1)+Bx(x+1)+Cx(x-1). \]

Use the zeros of the factors:

\(x=0\)\(1=-A \Rightarrow A=-1\)
\(x=1\)\(3=2B \Rightarrow B=\frac32\)
\(x=-1\)\(3=2C \Rightarrow C=\frac32\)

Therefore,

\[ \boxed{ \frac{2x^2+1}{x(x-1)(x+1)} =-\frac1x+\frac{3/2}{x-1}+\frac{3/2}{x+1} }. \]

Its indefinite integral is

\[ \boxed{-\ln|x|+\frac32\ln|x-1|+\frac32\ln|x+1|+C}. \]

4. Repeated Linear Factors

Every power of a repeated linear factor must appear in the decomposition.

REPEATED LINEAR-FACTOR FORM

For a factor \((x-r)^m\), include

\[ \boxed{ \frac{A_1}{x-r}+\frac{A_2}{(x-r)^2}+\cdots+ \frac{A_m}{(x-r)^m} }. \]

Skipping an intermediate power makes the decomposition incomplete.

EXAMPLE 3: A REPEATED FACTOR

Evaluate \[ \int\frac{3x+5}{x(x+1)^2}\,dx. \]

Use the complete form \[ \frac{3x+5}{x(x+1)^2} =\frac{A}{x}+\frac{B}{x+1}+\frac{C}{(x+1)^2}. \]

After multiplying by \(x(x+1)^2\), \[ 3x+5=A(x+1)^2+Bx(x+1)+Cx. \]

Set \(x=0\) to obtain \(A=5\). Set \(x=-1\) to obtain \(2=-C\), so \(C=-2\). Comparing the \(x^2\)-coefficients gives \(A+B=0\), so \(B=-5\). Hence,

\[ \begin{aligned} \int\frac{3x+5}{x(x+1)^2}\,dx &=\int\left(\frac5x-\frac5{x+1}-\frac2{(x+1)^2}\right)dx\\[3pt] &=\boxed{5\ln|x|-5\ln|x+1|+\frac2{x+1}+C}. \end{aligned} \]

5. Improper Rational Functions

EXAMPLE 4: LONG DIVISION BEFORE DECOMPOSITION

Evaluate \[ \int\frac{x^3+2x}{x^2-1}\,dx. \]

Long division gives \[ \frac{x^3+2x}{x^2-1}=x+\frac{3x}{x^2-1}. \]

The remainder may be decomposed:

\[ \frac{3x}{(x-1)(x+1)} =\frac{3/2}{x-1}+\frac{3/2}{x+1}. \]

Therefore,

\[ \boxed{ \int\frac{x^3+2x}{x^2-1}\,dx =\frac{x^2}{2}+\frac32\ln|x-1|+\frac32\ln|x+1|+C }. \]

Equivalently, the logarithms may be combined as \(\frac32\ln|x^2-1|\).

6. Verification

There are two useful checks.

ALGEBRAIC AND CALCULUS CHECKS
  • Recombine the partial fractions and confirm that the original rational function is recovered.
  • Differentiate the final antiderivative and confirm that the original integrand is recovered.

For Example 1,

\[ \frac2{x-1}+\frac3{x+2} =\frac{2(x+2)+3(x-1)}{(x-1)(x+2)} =\frac{5x+1}{(x-1)(x+2)}. \]
WARNING // COMMON ERRORS
  • Decomposing before making the fraction proper.
  • Failing to factor the denominator completely.
  • Omitting powers associated with a repeated linear factor.
  • Using a polynomial numerator over a linear factor; the numerator should be a constant.
  • Omitting absolute values in logarithmic antiderivatives.

7. Principal Results

LINEAR PARTIAL-FRACTION TEMPLATES
\[ \frac{P(x)}{(x-r)(x-s)} =\frac{A}{x-r}+\frac{B}{x-s} \]

and

\[ \frac{P(x)}{(x-r)^m} =\frac{A_1}{x-r}+\frac{A_2}{(x-r)^2}+\cdots+ \frac{A_m}{(x-r)^m}. \]

Make the fraction proper, factor completely, include every required term, solve for the constants, integrate, and verify.
