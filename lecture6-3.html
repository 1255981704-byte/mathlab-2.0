A Reliable Integration Workflow

WHAT ARE WE FINDING?

An indefinite integral is the complete family of antiderivatives.

DEFINITION
\[\displaystyle F'(x)=f(x)\quad\Longrightarrow\quad\int f(x)\,dx=F(x)+C.\]

The constant \(C\) represents every vertical shift with the same derivative.

FIVE MOVES
01
Simplify or rewrite the integrand.
02
Separate sums and constant multiples.
03
Match each term to a formula.
04
Include one constant \(C\).
05
Differentiate to verify.
FORMULAS

Algebraic Antiderivatives

CONSTANT & POWER
CONSTANT
\[\displaystyle \int k\,dx=kx+C\]
POWER RULE // N ≠ -1
\[\displaystyle \int x^n\,dx=\frac{x^{n+1}}{n+1}+C\]

Increase the exponent by one, then divide by the new exponent.

EXCEPTION & EXAMPLE
EXCEPTION // N = -1
\[\displaystyle \int\frac1x\,dx=\ln|x|+C\]
NEGATIVE POWER
\[\displaystyle \int x^{-4}\,dx=\frac{x^{-3}}{-3}+C=-\frac{1}{3x^3}+C.\]
Never apply the power rule to \(x^{-1}\).
FORMULAS

Exponential Antiderivatives

FORMULA & EXAMPLE
BASE E
\[\displaystyle \int e^x\,dx=e^x+C\]
BASE A
\[\displaystyle \int a^x\,dx=\frac{a^x}{\ln a}+C,\quad a>0,\ a\ne1\]

The denominator \(\ln a\) reverses the factor created by differentiating \(a^x\).

EVALUATE
\[\displaystyle \int5\cdot2^x\,dx=5\int2^x\,dx=\frac{5\cdot2^x}{\ln2}+C.\]

Basic Trigonometric Antiderivatives

SINE, COSINE & SQUARES
COSINE\[\int\cos x\,dx=\sin x+C\]
SINE\[\int\sin x\,dx=-\cos x+C\]
SECANT SQUARED\[\int\sec^2x\,dx=\tan x+C\]
COSECANT SQUARED\[\int\csc^2x\,dx=-\cot x+C\]
PRODUCT PAIRS
SECANT–TANGENT
\[\displaystyle \int\sec x\tan x\,dx=\sec x+C\]
COSECANT–COTANGENT
\[\displaystyle \int\csc x\cot x\,dx=-\csc x+C\]

Negative signs come directly from the corresponding derivative formulas.

PATTERN

Linear Inner Functions

REVERSE CHAIN RULE

For a linear inner function \(ax+b\), divide by its derivative \(a\).

POWER
\[\displaystyle \int(ax+b)^n dx=\frac{(ax+b)^{n+1}}{a(n+1)}+C\]
LOGARITHM
\[\displaystyle \int\frac1{ax+b}dx=\frac1a\ln|ax+b|+C\]
EXPONENTIAL & TRIG
\[\displaystyle \int e^{ax+b}dx=\frac1a e^{ax+b}+C\]
\[\displaystyle \int\cos(ax+b)dx=\frac1a\sin(ax+b)+C\]
\[\displaystyle \int\sin(ax+b)dx=-\frac1a\cos(ax+b)+C\]

03

Use an Initial Condition

PROBLEM
Find \(F\) given
DERIVATIVE & CONDITION
\[\displaystyle F'(x)=3x^2-4,\qquad F(1)=2.\]

An initial condition selects one function from the family \(F(x)+C\).

SOLUTION
1 // GENERAL ANTIDERIVATIVE
\[\displaystyle F(x)=x^3-4x+C.\]
2 // USE THE CONDITION
\[\displaystyle 2=1^3-4(1)+C=-3+C\quad\Rightarrow\quad C=5.\]
3 // PARTICULAR SOLUTION
\[\displaystyle \boxed{F(x)=x^3-4x+5}.\]
SUMMARY
