METHOD

Evaluate a Definite Integral

FTC EVALUATION & WORKFLOW

If \(F'(x)=f(x)\), then

FUNDAMENTAL THEOREM
\[\displaystyle \int_a^b f(x)\,dx=\left[F(x)\right]_a^b=F(b)-F(a).\]

The answer is a number representing net accumulation over ([a,b]).

A definite integral does not include (+C).

SYMMETRY

Evaluate Before You Integrate

EVEN & ODD FUNCTIONS
EVEN // MIRROR SYMMETRY\[\displaystyle f(-x)=f(x),\qquad \int_{-a}^{a}f(x)\,dx=2\int_0^a f(x)\,dx\]
ODD // ROTATIONAL SYMMETRY\[\displaystyle f(-x)=-f(x),\qquad \int_{-a}^{a}f(x)\,dx=0\]

These shortcuts require a symmetric interval ([-a,a]).

GEOMETRY
Evaluate without finding an antiderivative:
UPPER SEMICIRCLE
\[\displaystyle \int_{-3}^{3}\sqrt{9-x^2}\,dx\]
RADIUS 3
\[\displaystyle \frac12\pi(3)^2=\frac{9\pi}{2}.\]
DEFINITION

Average Value of a Function

AVERAGE VALUE

If \(f\) is continuous on \([a,b]\), its average value is

FORMULA
\[\displaystyle \boxed{f_{\mathrm{avg}}=\frac{1}{b-a}\int_a^b f(x)\,dx}\]

It is the constant height of a rectangle with the same signed area as the region under \(f\).

The divisor is interval length \(b-a\), not the number of endpoints.

EQUAL-AREA PICTURE
FUNCTION AND AVERAGE HEIGHT
\[\displaystyle (b-a)f_{\mathrm{avg}}=\int_a^b f(x)\,dx.\]
02

Find an Average Value

PROBLEM & SOLUTION
Find the average value of \(f(x)=x^2+1\) on \([0,3]\).
SETUP TEMPLATE
\[\displaystyle f_{\mathrm{avg}}=\frac{1}{\text{interval length}}\int_{\text{left}}^{\text{right}}f(x)\,dx\]
INTERVAL LENGTH\(3-0=3\)
ANTIDERIVATIVE\(x^3/3+x\)
WORKED SOLUTION
\[\displaystyle f_{\mathrm{avg}}=\frac13\int_0^3(x^2+1)\,dx\]
\[\displaystyle =\frac13\left[\frac{x^3}{3}+x\right]_0^3=\frac13(12)=4.\]
By the Mean Value Theorem for Integrals, \(f(c)=4\) for at least one \(c\in[0,3]\).

THEOREM

Net Change: Rate to Quantity

NET CHANGE THEOREM

If \(Q'(t)\) is the rate of change of \(Q\), then

NET CHANGE
\[\displaystyle \int_a^b Q'(t)\,dt=Q(b)-Q(a).\]

To recover the final amount, include the initial amount:

FINAL QUANTITY
\[\displaystyle \boxed{Q(b)=Q(a)+\int_a^b Q'(t)\,dt}\]
APPLICATION
Water enters a tank at \(r(t)=6+2t\) L/min. Initially the tank holds \(40\) L. Find the amount after \(5\) min.
\[\displaystyle Q(5)=40+\int_0^5(6+2t)\,dt\]
\[\displaystyle =40+[6t+t^2]_0^5=95\text{ L}.\]
UNIT CHECK\((\text{liters/minute})(\text{minutes})=\text{liters}\)
MOTION

Displacement versus Total Distance

TWO QUESTIONS
DISPLACEMENT // NET CHANGE IN POSITION\[\displaystyle \int_a^b v(t)\,dt=s(b)-s(a)\]
TOTAL DISTANCE // ALL MOTION POSITIVE\[\displaystyle \int_a^b|v(t)|\,dt\]

Split the interval wherever velocity changes sign.

EXAMPLE
Let \(v(t)=(t-1)(t-3)\) on \(0\le t\le4\).
Velocity changes sign at \(t=1\) and \(t=3\).
\[\displaystyle \text{displacement}=\int_0^4v(t)\,dt=\frac43\]
\[\displaystyle \text{distance}=\int_0^1v-\int_1^3v+\int_3^4v=4\]
Absolute value prevents backward motion from canceling forward motion.
